# API Testing Project – FakeStore API

## Project Description
This project demonstrates API testing using Postman on the FakeStore REST API.

## API Requests Performed
- GET Products
- GET Single Product
- POST Create Product
- PUT Update Product
- DELETE Product

## Validations
- HTTP Status Code Validation
- Response Body Validation
- Positive Testing
- Negative Testing

## Tools Used
- Postman
- REST API
- JSON
