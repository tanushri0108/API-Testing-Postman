# API Testing Projects using Postman

## Project Overview
This repository contains API Testing projects created using Postman on public REST APIs.

## Project 1 – ReqRes API
### API Requests Performed
- GET Users
- GET Single User
- POST Login
- POST Create User
- PUT Update User
- DELETE User

### Validations
- HTTP Status Code Validation
- Response Body Validation
- Positive Testing
- Negative Testing
